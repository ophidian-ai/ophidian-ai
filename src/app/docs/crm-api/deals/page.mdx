# Deals API

Manage deals in your CRM pipeline. Growth tier has read access; Pro tier has full read/write access.

---

## List Deals

`GET /api/crm/[slug]/deals`

Returns deals for your CRM config. Supports optional filters.

### Query Parameters

| Parameter | Type | Description |
|---|---|---|
| `pipeline_id` | string | Filter by pipeline ID |
| `stage` | string | Filter by stage name (e.g., `Proposal`) |
| `limit` | number | Max results (default 50, max 200) |
| `offset` | number | Pagination offset (default 0) |

### Response

```json
{
  "deals": [
    {
      "id": "uuid",
      "config_id": "uuid",
      "pipeline_id": "uuid",
      "contact_id": "uuid",
      "title": "Acme Corp Website",
      "value": 2500,
      "stage": "Proposal",
      "probability": 60,
      "source": "chatbot",
      "expected_close_at": "2026-04-01T00:00:00Z",
      "won_at": null,
      "lost_at": null,
      "created_at": "2026-03-01T00:00:00Z",
      "updated_at": "2026-03-15T00:00:00Z"
    }
  ],
  "limit": 50,
  "offset": 0
}
```

### cURL

```bash
curl "https://ophidianai.com/api/crm/your-slug/deals?stage=Proposal" \
  -H "Authorization: Bearer YOUR_API_KEY"
```

### JavaScript

```js
const res = await fetch(
  "https://ophidianai.com/api/crm/your-slug/deals?stage=Proposal",
  {
    headers: { Authorization: "Bearer YOUR_API_KEY" },
  }
);
const { deals } = await res.json();
```

## Create Deal

`POST /api/crm/[slug]/deals`

**Pro tier only.** Creates a new deal.

### Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `pipeline_id` | string | Yes | ID of the pipeline |
| `contact_id` | string | Yes | ID of the contact |
| `title` | string | Yes | Deal title |
| `stage` | string | Yes | Stage name (must match pipeline stages) |
| `value` | number | No | Deal value in dollars |
| `source` | string | No | Lead source (default: `"api"`) |
| `expected_close_at` | string | No | ISO 8601 date |
| `custom_field_values` | object | No | Custom field key/value pairs |

### Response

Returns `201` with the created deal.

```json
{
  "deal": {
    "id": "uuid",
    "title": "New Website Project",
    "stage": "Lead",
    "value": 1500,
    ...
  }
}
```

### cURL

```bash
curl -X POST https://ophidianai.com/api/crm/your-slug/deals \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "pipeline_id": "pipeline-uuid",
    "contact_id": "contact-uuid",
    "title": "New Website Project",
    "stage": "Lead",
    "value": 1500,
    "source": "referral"
  }'
```

### JavaScript

```js
const res = await fetch("https://ophidianai.com/api/crm/your-slug/deals", {
  method: "POST",
  headers: {
    Authorization: "Bearer YOUR_API_KEY",
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    pipeline_id: "pipeline-uuid",
    contact_id: "contact-uuid",
    title: "New Website Project",
    stage: "Lead",
    value: 1500,
  }),
});
const { deal } = await res.json();
```

## Update Deal Stage

`PATCH /api/crm/[slug]/deals/[id]`

**Pro tier only.** Updates the stage of a deal. Triggers automations if configured.

### Request Body

| Field | Type | Required | Description |
|---|---|---|---|
| `stage` | string | Yes | New stage name (must exist in the deal's pipeline) |

### Response

Returns the updated deal.

```json
{
  "deal": {
    "id": "uuid",
    "stage": "Won",
    "probability": 100,
    "won_at": "2026-03-19T12:00:00Z",
    ...
  }
}
```

### cURL

```bash
curl -X PATCH https://ophidianai.com/api/crm/your-slug/deals/deal-uuid \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{"stage": "Won"}'
```

### JavaScript

```js
const res = await fetch(
  "https://ophidianai.com/api/crm/your-slug/deals/deal-uuid",
  {
    method: "PATCH",
    headers: {
      Authorization: "Bearer YOUR_API_KEY",
      "Content-Type": "application/json",
    },
    body: JSON.stringify({ stage: "Won" }),
  }
);
const { deal } = await res.json();
```
