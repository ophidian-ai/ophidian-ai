# OphidianAI Website

## Overview
Company website for OphidianAI. Marketing site with service descriptions, pricing, portfolio, blog, and contact form.

## Status
- **Phase:** Live (post-launch)
- **URL:** ophidian-ai.vercel.app
- **Custom Domain:** Not yet connected

## Tech Stack
- Next.js 15, React 19, TypeScript, Tailwind CSS 4, App Router
- MDX blog, Resend email, Vercel Analytics
- Deployed on Vercel (auto-deploy on push to main)

## Codebase
The website source code lives at the **repo root** (`src/`, `public/`, config files). This is an internal project -- the codebase and the Iris operational repo share the same repository.
