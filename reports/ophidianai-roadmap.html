Phase 1: Immediate
Required before full production launch
These items must be completed before directing clients to the site. The contact form is the primary lead capture mechanism. Without it, the site cannot convert visitors into leads.
1. Set Up Resend + RESEND_API_KEY
CRITICAL
Create a Resend account at resend.com. Generate an API key. Add RESEND_API_KEY to Vercel project environment variables. The contact form server action is already wired -- this is the only missing piece.
Without this, every contact form submission silently fails. No leads captured.
2. Connect Custom Domain
CRITICAL
Add ophidianai.com (or chosen domain) to the Vercel project. Update DNS records (CNAME or A record) at the domain registrar. Vercel handles SSL automatically. Update the metadataBase URL in the root layout once the domain is live.
Professional domain builds credibility. Required for client-facing outreach.
3. Verify Resend Domain
HIGH
After connecting the domain, verify it in Resend so contact form emails come from contact@ophidianai.com instead of onboarding@resend.dev. Requires adding DNS records (DKIM, SPF). Prevents emails from landing in spam.
Emails from a verified domain have much higher deliverability and look professional.

Phase 2: Near-Term
Revenue enablement and content -- next 30 days
1. Stripe Subscription Checkout
HIGH
Wire Stripe Checkout for Starter ($2,000/mo) and Growth ($3,500/mo) tiers. Create Stripe Products and Prices. Add API route for creating checkout sessions. Enterprise tier links to contact form or Calendly. Update pricing page CTAs to trigger checkout.
Enables self-serve revenue. Clients can sign up without a sales call for standard tiers.
2. Blog Content Sprint
HIGH
Publish 4-5 more blog posts covering: AI integration use cases for small businesses, case study deep-dive on Bloomin' Acres, "what to expect" from working with an AI agency, industry trends, and common automation opportunities. Each post is an MDX file in src/content/blog/.
SEO authority. Each post is a potential organic search entry point. Demonstrates expertise to prospects.
3. Calendly Integration
HIGH
Create Calendly account. Add embedded booking widget on the Contact page and Pricing page (Enterprise tier). Lazy-load the embed for performance. Provides a lower-friction alternative to the contact form for prospects ready to talk.
Reduces friction for high-intent prospects. Enterprise leads can book directly.
4. Privacy Policy + Terms of Service
MEDIUM
Create /privacy and /terms pages with appropriate legal content. Required for Stripe integration and general business compliance. Add links back to the footer once pages exist.
Legal requirement for payment processing. Builds trust with prospects.
5. Portfolio Expansion
MEDIUM
Add more detail to the Bloomin' Acres case study: before/after screenshots, key metrics, technologies used, timeline, client testimonial (if available). As new projects complete, add them to the portfolio grid.
Portfolio depth is the #1 trust factor for prospective clients evaluating an agency.
6. Social Media Links
MEDIUM
Set up LinkedIn company page and professional profile. Once active, add social links back to the footer. Consider an X/Twitter presence for industry visibility. GitHub link already present.
Social presence validates the business. LinkedIn is critical for B2B credibility.

Phase 3: Future
Growth features and polish -- backlog
1. Client Portal
MEDIUM
Authenticated area where active clients can view project status, deliverables, invoices, and communication history. Integrate with Supabase Auth. Could replace or complement ClickUp for client-facing project visibility.
Professional client experience. Reduces back-and-forth emails about project status.
2. Newsletter / Email List
MEDIUM
Add email signup form (footer or dedicated section). Integrate with Resend or a dedicated newsletter service. Use for blog post announcements, AI industry updates, and promotional campaigns.
Owned audience. Direct channel to prospects who aren't ready to buy yet.
3. Testimonials Section
MEDIUM
Add a TestimonialCarousel component to the home page. Collect quotes from Bloomin' Acres and future clients. Display with name, company, and headshot. Social proof is one of the strongest conversion drivers.
Testimonials can increase conversion by 20-30% on service pages.
4. Founder Photo
LOW
Replace the styled EL monogram on the About page with a professional headshot. Personal photos on agency sites increase trust and humanize the brand.
People buy from people. A face builds connection.
5. Favicon Variants
LOW
Add apple-touch-icon.png and favicon-32x32.png for better device coverage. Currently only favicon.ico exists. Low effort, marginally improves appearance on iOS home screen and browser tabs.
Small polish item. Improves appearance on Apple devices.
6. Blog Featured Images
LOW
Add featured image support to MDX frontmatter. Display on blog index cards and at the top of individual posts. Currently using a generic icon placeholder.
Visual blog cards get more clicks. Better social media sharing appearance.
7. Performance Monitoring
LOW
Set up Web Vitals monitoring via Vercel Analytics (already installed) or a dedicated service. Track LCP, FID, CLS across pages. Set up alerts for regressions. Run Lighthouse CI on deploys.
Catch performance regressions before they affect user experience or SEO rankings.
